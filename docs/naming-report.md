## 네이밍 기준

1. **발음 가능성** — 한국어, 영어, 일본어, 중국어, 아랍어, 힌디어 화자 모두 발음 가능
2. **상표 안전** — 수산/펫/앱 카테고리 유명 상표와 혼동 가능성 없음
3. **도메인 가용성** — .com 취득 가능성 (완전 일치 어려우면 .app 등 대안 허용)
4. **의미 중립/긍정** — 어느 언어권에서도 부정적 의미 없음
5. **기억용이성** — 2~3음절 선호, 최대 4음절
6. **확장성** — 어종 외 다른 반려동물로 확장 시 브랜드 훼손 없음
7. **시각화** — 로고/아이콘으로 표현 가능

## 최종 추천

### 🥇 1위: Finara

**어원**: Fin(지느러미) + *-ara*(라틴어 접미사, 공간·장소) = "지느러미들의 공간"

#### 언어별 발음

| 언어 | 표기 | 발음 |
|---|---|---|
| 한국어 | 피나라 | 피-나-라 |
| 영어 | Finara | fi-NAR-a |
| 일본어 | フィナーラ | fi-nā-ra |
| 중국어 | 芬纳拉 | Fēn-nà-lā (의미 중립) |
| 아랍어 | فينارا | Fīnārā |
| 힌디어 | फिनारा | Fināra |

#### 도메인

| 도메인 | 상태 | 비용 추정 |
|---|---|---|
| finara.com | 파킹 상태 (협상 필요) | $2,000~$15,000 |
| **finara.app** | **취득 가능 (권장)** | ~$20/년 |
| getfinara.com | 신규 등록 가능 | ~$12/년 |

#### 상표 리스크

- **종합: 중간-낮음**
- 체코 금융회사 Finova와 유사 철자 존재하나 "Finara"와 철자 상이 (Finova ≠ Finara)
- Class 35(플랫폼), 42(소프트웨어) 조기 출원 시 방어 가능
- **KIPRIS(한국): 낮음** — 비교적 클린

#### 슬로건 후보

1. *"Where every fin has a story."* — 모든 지느러미에는 이야기가 있다
2. *"The world's fish, in your hands."* — 세계의 물고기를 당신 손 위에
3. *"Discover. Connect. Belong."* — 발견하고, 연결하고, 속하라

#### 로고 방향

`F`자를 부드러운 지느러미 곡선으로 변형. 물방울 또는 수조 원형 프레임과 결합.
색상: **딥 오션 블루 + 빛나는 코랄 오렌지**.

#### 총평

Canva, Figma, Aura처럼 `-a`로 끝나는 글로벌 SaaS 브랜드 문법에 완벽 부합하는 3음절 구조.
9개 언어권에서 발음 가능하고 부정적 의미 없음.
상표 리스크는 조기 출원으로 관리 가능하며, 슬로건-로고-앱 아이콘 연계가 후보 중 가장 강력.

### 🥈 2위: Pinna

**어원**: 라틴어 "지느러미, 날개, 깃털" — 이탈리아어·스페인어에서도 동일 의미

#### 언어별 발음

| 언어 | 표기 | 발음 |
|---|---|---|
| 한국어 | 피나 | 피-나 (2음절, 깔끔) |
| 영어 | Pinna | PIN-na |
| 일본어 | ピンナ | Pin-na (귀엽고 자연스러움) |
| 중국어 | 品纳 | Pǐn-nà ("품격 있게 받아들이다" — 긍정적 의미) |
| 아랍어 | بينا | 약간 혼동 가능하나 발음 가능 |
| 힌디어 | पिन्ना | Pinna (2음절, 자연스러움) |

#### 도메인

| 도메인 | 상태 |
|---|---|
| pinna.com | 점유됨 (의료/교육 서비스) |
| **pinna.app** | 취득 가능 (권장) |
| pinna.io | 취득 가능성 있음 |

#### 상표 리스크

- **종합: 중간**
- 미국 청각 보조기기 회사 Pinna Inc. (Class 10) 존재
- 다른 클래스(35, 42)에서는 법적 공존 가능하나 투자·M&A 시 설명 필요
- KIPRIS(한국) 비교적 클린

#### 슬로건 후보

1. *"Life beneath the surface."* — 수면 아래의 삶
2. *"Small fins, big world."* — 작은 지느러미, 넓은 세상
3. *"Where fish find family."* — 물고기가 가족을 만나는 곳

#### 로고 방향

`P`자 자체를 둥근 수조 실루엣으로. 내부에 미니멀 물고기 한 마리.
2음절 최소구조로 워드마크 강력함.

#### 총평

2음절의 강력한 기억용이성과 라틴어 어원의 정통성이 최대 강점.
단, .com 도메인 확보 어려움과 의료기기 브랜드 공존 리스크가 약점.
도메인 전략을 .app으로 가져간다면 강력한 대안.

### 🥉 3위: Aqulio

**어원**: Aqua(물) + *-lio*(이탈리아어 접미사 연상) — 완전 신조어

#### 언어별 발음

| 언어 | 발음 | 비고 |
|---|---|---|
| 한국어 | 아-쿠-리-오 | 4음절, 약간 김 |
| 영어 | ah-KYOO-lee-oh | 발음 가이드 필요 (단점) |
| 일본어 | アクリオ | 수족관(アクアリウム) 연상 — 긍정적 |
| 중국어 | 阿酷里奥 | 4음절, 발음 가능하나 복잡 |

#### 도메인

- **aqulio.com**: 신조어로 즉시 등록 가능 (추정 $12)

#### 상표 리스크

- **종합: 낮음** — 완전 신조어, 현재 유사 상표 없음
- Class 35, 42 선출원 시 강력 보호 가능

#### 슬로건 후보

1. *"Your aquatic universe."*
2. *"Dive deeper."*
3. *"From tank to world."*

#### 총평

도메인 확보 용이성과 상표 클린함이 최대 강점.
그러나 4음절의 길이와 영어권 발음 불확실성이 약점.
도메인·상표 리스크를 최소화하고 싶을 때의 실용적 백업 옵션.
